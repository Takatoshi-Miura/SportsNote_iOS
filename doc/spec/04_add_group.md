# グループ追加画面

## 概要

新規グループを作成する画面。課題一覧画面の＋ボタンからsheet表示される。

## 画面構成

- ナビゲーションバー: タイトル「グループの追加」、「キャンセル」ボタン、「保存」ボタン
- GroupForm:
  - タイトル（TextField、必須）
  - カラー選択（LazyVGrid、8色のカラーサークル）

## 前提条件

- 課題一覧画面の＋ボタン →「グループ」選択で表示

## 機能仕様

### 入力

| 操作 | 期待結果 |
|------|----------|
| タイトルを入力 | テキストフィールドに入力値が反映される |
| カラーサークルをタップ | 選択した色にボーダーが付き、選択状態になる |

### カラー選択肢

| 色名 | 説明 |
|------|------|
| red | 赤 |
| pink | ピンク |
| orange | オレンジ |
| yellow | 黄色 |
| green | 緑 |
| blue | 青 |
| purple | 紫 |
| gray | グレー |

### 保存

| 操作 | 期待結果 |
|------|----------|
| タイトル入力済みで「保存」タップ | グループがRealmに保存され、画面が閉じる |
| タイトル未入力で「保存」タップ | 保存ボタンが無効化されているためタップ不可 |

### キャンセル

| 操作 | 期待結果 |
|------|----------|
| 「キャンセル」をタップ | 入力内容を破棄して画面が閉じる |

## バリデーション

| 項目 | ルール |
|------|--------|
| タイトル | 必須。空の場合、保存ボタンが無効化 |
| カラー | 必須（デフォルト: red） |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| 保存成功 | 課題一覧画面 | sheet dismiss |
| キャンセル | 課題一覧画面 | sheet dismiss |

## 補足

- order値は自動計算される（既存グループ数をもとに設定）
- 保存時にFirebaseへバックグラウンド同期される
