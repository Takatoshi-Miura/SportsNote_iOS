# 利用規約画面

## 概要

利用規約への同意を求めるダイアログ。アプリ起動時に未同意の場合に自動表示される。

## 画面構成

- アラートダイアログ:
  - タイトル: 利用規約
  - メッセージ: 利用規約への同意を求めるテキスト
  - ボタン:
    - 「利用規約を確認する」
    - 「同意する」

## 表示条件

- UserDefaultsの`agree`フラグがfalseの場合に表示
- MainTabView表示時（onAppear）およびアプリがフォアグラウンド復帰時に確認

## 機能仕様

### ダイアログ操作

| 操作 | 期待結果 |
|------|----------|
| 「利用規約を確認する」をタップ | Safariで利用規約ページが開く |
| 「同意する」をタップ | UserDefaultsのagreeフラグがtrueに設定され、ダイアログが閉じる |

## 表示タイミング

| タイミング | 条件 |
|-----------|------|
| アプリ起動時 | agree = false の場合 |
| フォアグラウンド復帰時 | agree = false の場合 |

## 補足

- 同意するまでダイアログは表示され続ける
- 一度同意すると、以降は表示されない
