# ノート一覧画面

## 概要

ノートタブのメイン画面。フリーノート・練習ノート・大会ノートを一覧表示し、検索・削除を行う。

## 画面構成

- ナビゲーションバー: タイトル「ノート」、左にメニューボタン、右にページボタン（doc.plaintext）
- 検索バー: テキスト入力フィールド（虫眼鏡アイコン + クリアボタン）
- ノートリスト: 全ノート一覧（List）
- 右下に＋ボタン（FAB）
- 画面下部: AdMobバナー広告

## 前提条件

- アプリ初期化済み
- フリーノートが1件存在する（初回起動時に自動作成）

## 機能仕様

### ノートリスト

| ノート種別 | アイコン | インジケーター色 | 行の表示内容 |
|-----------|---------|----------------|-------------|
| フリーノート | doc.text | グレー | タイトル + 詳細テキスト |
| 練習ノート | figure.run | グループカラー（課題に基づく） | 練習内容 + 日付 |
| 大会ノート | trophy | オレンジ（固定） | 結果 + 日付 |

### ノートリスト操作

| 操作 | 期待結果 |
|------|----------|
| ノート行をタップ | ノート種別に応じた詳細画面に遷移する |
| 練習/大会ノート行を左スワイプ | 削除ボタンが表示される |
| 削除ボタンをタップ | ノートが論理削除され、リストから消える |
| フリーノート行を左スワイプ | 削除ボタンは表示されない（削除不可） |
| リストを下に引っ張る | データが再取得される（Pull to Refresh） |

### 検索機能

| 操作 | 期待結果 |
|------|----------|
| 検索バーにテキスト入力 | リアルタイムでノートがフィルタリングされる |
| 検索結果が0件の場合 | 「ノートが見つかりません」メッセージが表示される |
| 検索バーのクリアボタンをタップ | 検索テキストがクリアされ、全ノートが表示される |

### ページボタン

| 操作 | 期待結果 |
|------|----------|
| ページボタンをタップ | ノートページ画面に遷移する |
| フリーノート以外のノートが存在しない場合 | ページボタンが無効化される |

### ＋ボタン

| 操作 | 期待結果 |
|------|----------|
| ＋ボタンをタップ | アクションシート「追加」が表示される |
| 「練習ノート」を選択 | 練習ノート追加画面がsheet表示される |
| 「大会ノート」を選択 | 大会ノート追加画面がsheet表示される |
| 「キャンセル」を選択 | アクションシートが閉じる |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| フリーノート行タップ | フリーノート画面 | NavigationLink |
| 練習ノート行タップ | 練習ノート詳細画面 | NavigationLink |
| 大会ノート行タップ | 大会ノート詳細画面 | NavigationLink |
| ページボタンタップ | ノートページ画面 | NavigationLink |
| ＋ボタン →「練習ノート」 | 練習ノート追加画面 | sheet |
| ＋ボタン →「大会ノート」 | 大会ノート追加画面 | sheet |
| メニューボタンタップ | メニュー画面 | サイドメニュー |

## 補足

- 検索はノートのテキストフィールドとメモの内容を対象とする（大文字小文字区別なし）
- フリーノートは削除不可（swipeActions未適用）
- フリーノートは常に1件存在するため、通常時にリストが空になることはない
- ノート削除時に関連するメモもカスケード論理削除される
