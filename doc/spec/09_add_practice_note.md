# 練習ノート追加画面

## 概要

練習ノートを新規作成する画面。基本情報・練習内容・取り組んだ課題を入力する。

## 画面構成

- ナビゲーションバー: タイトル「練習ノートの追加」、「キャンセル」ボタン、「保存」ボタン
- Form:
  - 基本情報セクション（BasicInfoSection）: 日付、天気、気温
  - 体調（TextEditorSection）
  - 目的（TextEditorSection）
  - 練習内容（TextEditorSection）
  - 取り組んだ課題（TaskListSection）
  - 反省（TextEditorSection）

## 前提条件

- ノート一覧画面の＋ボタン →「練習ノート」選択で表示

## 機能仕様

### 基本情報入力

| 項目 | 入力方法 | デフォルト値 |
|------|---------|-------------|
| 日付 | DatePicker | 今日の日付 |
| 天気 | Picker（sunny/cloudy/rainy） | sunny |
| 気温 | Stepper（℃） | 20 |

### テキスト入力

| 項目 | 入力方法 | 必須 |
|------|---------|------|
| 体調 | AutoResizingTextEditor | 任意 |
| 目的 | AutoResizingTextEditor | 任意 |
| 練習内容 | AutoResizingTextEditor | 任意 |
| 反省 | AutoResizingTextEditor | 任意 |

### 取り組んだ課題セクション

| 操作 | 期待結果 |
|------|----------|
| 画面表示時 | 未完了かつ対策が設定されている課題が自動追加される |
| 「課題を追加」ボタンをタップ | 課題選択ダイアログが表示される |
| 課題選択ダイアログで課題をタップ | 選択した課題が取り組み課題リストに追加される |
| 課題行のメニュー（…）ボタンをタップ | 削除確認アラートが表示される |
| 削除確認で「削除」 | 課題がリストから削除される |
| 各課題の反省テキストを入力 | テキストが保持される |
| 未追加課題が0件の場合 | 「課題を追加」ボタンが無効化される |

### 保存

| 操作 | 期待結果 |
|------|----------|
| 「保存」をタップ | 練習ノートと課題反省メモがRealmに保存され、画面が閉じる |

### キャンセル

| 操作 | 期待結果 |
|------|----------|
| 「キャンセル」をタップ | 入力内容を破棄して画面が閉じる |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| 保存成功 | ノート一覧画面 | sheet dismiss |
| キャンセル | ノート一覧画面 | sheet dismiss |
| 「課題を追加」タップ | 課題選択ダイアログ | sheet |

## 補足

- 保存時にFirebaseへバックグラウンド同期される
- 課題反省テキストはMemoエンティティとして保存される（measuresIDとnoteIDで紐づけ）
- 全項目が任意入力（バリデーションなし）
