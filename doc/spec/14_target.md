# 目標一覧画面

## 概要

目標タブのメイン画面。カレンダーで日付を選択し、年間/月間目標を確認する。選択日のノートも表示する。

## 画面構成

- ナビゲーションバー: タイトル「目標」、左にメニューボタン、右に「今日」ボタン
- カレンダーセクション（固定）:
  - 目標サマリー（TargetSummaryView）: 年間目標と月間目標のタイトル
  - カレンダー（CalendarView）: 月表示カレンダー
- ノートリストセクション（スクロール可能、日付選択時のみ表示）
- 右下に＋ボタン（FAB）
- 画面下部: AdMobバナー広告

## 前提条件

- アプリ初期化済み

## 機能仕様

### カレンダー操作

| 操作 | 期待結果 |
|------|----------|
| 日付セルをタップ | その日付が選択され、ノートリストセクションが表示される |
| 左スワイプ | 翌月のカレンダーが表示される |
| 右スワイプ | 前月のカレンダーが表示される |
| 「今日」ボタンをタップ | 今日の日付が選択され、現在の月に移動する |

### カレンダー表示

| 表示要素 | 条件 |
|---------|------|
| 日曜日の日付 | 赤色で表示 |
| 土曜日の日付 | 青色で表示 |
| 日本の祝日 | 赤色で表示 |
| ノートがある日付 | インジケーター（色付きドット）が表示される |
| 今日の日付 | 背景色でハイライト表示 |

### 目標サマリー

| 表示内容 | 説明 |
|---------|------|
| 年間目標 | 選択中の年の年間目標タイトル（未設定の場合は「未設定」） |
| 月間目標 | 選択中の月の月間目標タイトル（未設定の場合は「未設定」） |

### ノートリスト（日付選択時）

| 操作 | 期待結果 |
|------|----------|
| ノート行をタップ | ノート種別に応じた詳細画面に遷移する |

### ＋ボタン

| 操作 | 期待結果 |
|------|----------|
| ＋ボタンをタップ | アクションシート「追加」が表示される |
| 「年間目標」を選択 | 目標追加画面がsheet表示される（年間目標モード） |
| 「月間目標」を選択 | 目標追加画面がsheet表示される（月間目標モード） |
| 「キャンセル」を選択 | アクションシートが閉じる |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| ノート行タップ | ノート詳細画面（種別に応じて） | NavigationLink |
| ＋ボタン →「年間目標」 | 目標追加画面 | sheet |
| ＋ボタン →「月間目標」 | 目標追加画面 | sheet |
| メニューボタンタップ | メニュー画面 | サイドメニュー |

## 補足

- カレンダーは日本の祝日に対応（JapaneseHolidayChecker）
- 年月変更時に目標が自動的に再取得される（Combineで監視）
- ノートリストはフリーノートを除外して表示
- 日付未選択時はノートリストセクションが非表示
