# ノートページ画面

## 概要

練習ノート・大会ノートを読み取り専用でページング表示する画面。スワイプでノート間を移動できる。

## 画面構成

- ナビゲーションバー: ノート種別アイコン + ノート種別名（動的に切り替わる）
- TabView（PageTabViewStyle）: スワイプでノート切り替え
- 各ページ（NotePageContentView）: 読み取り専用のノート内容

## 前提条件

- ノート一覧画面のページボタンタップで遷移
- フリーノート以外のノートが1件以上存在する

## 機能仕様

### ページ操作

| 操作 | 期待結果 |
|------|----------|
| 左スワイプ | 次のノートページが表示される |
| 右スワイプ | 前のノートページが表示される |
| ページインジケーター | 現在のページ位置が表示される |

### ナビゲーションバー表示

| 条件 | 表示 |
|------|------|
| 練習ノート表示中 | figure.runアイコン + 「練習ノート」 |
| 大会ノート表示中 | trophyアイコン + 「大会ノート」 |

### ページ内容（練習ノート）

| セクション | 表示内容 |
|-----------|---------|
| 基本情報 | 日付（曜日付き）、天気アイコン、気温 |
| 体調 | テキスト（未入力の場合は「-」） |
| 目的 | テキスト |
| 練習内容 | テキスト |
| 取り組んだ課題 | 課題名 + 反省テキスト（0件の場合は「取り組んだ課題はありません」） |
| 反省 | テキスト |

### ページ内容（大会ノート）

| セクション | 表示内容 |
|-----------|---------|
| 基本情報 | 日付（曜日付き）、天気アイコン、気温 |
| 体調 | テキスト（未入力の場合は「-」） |
| 目標 | テキスト |
| 意識すること | テキスト |
| 結果 | テキスト |
| 反省 | テキスト |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| 戻るボタン | ノート一覧画面 | NavigationStack pop |

## 補足

- フリーノートは表示対象外（fetchNotesExcludingFreeで取得）
- 全フィールドは読み取り専用（編集不可）
- 画面表示時にタブバーが非表示になる
