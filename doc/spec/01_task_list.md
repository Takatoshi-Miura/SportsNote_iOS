# 課題一覧画面

## 概要

課題タブのメイン画面。グループごとに課題を管理し、フィルタリング・並び替え・完了切り替えを行う。

## 画面構成

- ナビゲーションバー: タイトル「課題」、左にメニューボタン、右に並び替えボタン・フィルタボタン
- グループリストセクション: 水平スクロール可能なグループチップ一覧
- 課題リスト: フィルタされた課題一覧（List）
- 右下に＋ボタン（FAB）
- 画面下部: AdMobバナー広告

## 前提条件

- アプリ初期化済み
- グループが1つ以上存在する（初回起動時に「未分類」グループが自動作成）

## 機能仕様

### グループチップ操作

| 操作 | 期待結果 |
|------|----------|
| グループチップをタップ | 選択したグループの課題のみ表示される |
| 選択中のグループチップを再タップ | フィルタが解除され、全グループの課題が表示される |
| グループチップのiアイコンをタップ | グループ詳細画面に遷移する |

### 課題リスト操作

| 操作 | 期待結果 |
|------|----------|
| 課題行をタップ | 課題詳細画面に遷移する |
| 課題行を右スワイプ | 完了/未完了の切り替えボタンが表示される |
| 完了切り替えボタンをタップ | 課題の完了状態が反転する |
| 課題行を左スワイプ | 削除ボタンが表示される |
| 削除ボタンをタップ | 削除確認アラートが表示される |
| 削除確認で「削除」をタップ | 課題が論理削除される |
| リストを下に引っ張る | データが再取得される（Pull to Refresh） |

### 課題行の表示内容

| 表示要素 | 説明 |
|---------|------|
| グループカラーサークル | 課題が属するグループの色 |
| 課題タイトル | 課題のタイトルテキスト（完了済みは取り消し線） |
| 優先対策 | order値が最小の対策タイトル |

### フィルタ操作

| 操作 | 期待結果 |
|------|----------|
| フィルタボタンをタップ | メニューが表示される |
| 「完了済みの課題を表示」を選択 | 完了済み課題がリストに追加表示される |
| 「完了済みの課題を非表示」を選択 | 完了済み課題がリストから非表示になる |

### 並び替え操作

| 操作 | 期待結果 |
|------|----------|
| 並び替えボタンをタップ | 並び替えモードに切り替わる（ドラッグハンドル表示） |
| 課題行をドラッグ＆ドロップ | 課題の表示順序が変更され、order値が更新される |
| 並び替えボタンを再タップ | 通常モードに戻る |

### 並び替えモードの制約

| 条件 | 動作 |
|------|------|
| グループが選択されている場合 | 並び替えボタンが無効化される |
| 並び替えモード中 | フィルタボタンとグループチップが無効化される |
| 並び替えモード中 | ＋ボタンが非表示になる |

### ＋ボタン

| 操作 | 期待結果 |
|------|----------|
| ＋ボタンをタップ | アクションシート「追加」が表示される |
| 「グループ」を選択 | グループ追加画面がsheet表示される |
| 「課題」を選択 | 課題追加画面がsheet表示される |
| 「キャンセル」を選択 | アクションシートが閉じる |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| 課題行タップ | 課題詳細画面 | NavigationLink |
| グループチップのiアイコンタップ | グループ詳細画面 | NavigationDestination |
| ＋ボタン →「グループ」 | グループ追加画面 | sheet |
| ＋ボタン →「課題」 | 課題追加画面 | sheet |
| メニューボタンタップ | メニュー画面 | サイドメニュー |

## 補足

- デフォルトでは完了済み課題は非表示
- 課題削除は論理削除（isDeleted = true）で、関連する対策・メモもカスケード論理削除される
- 画面表示時に毎回最新データを取得する
