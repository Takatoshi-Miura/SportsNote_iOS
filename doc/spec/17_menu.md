# メニュー画面

## 概要

各タブ共通のサイドメニュー。ログイン、ヘルプ、利用規約などの設定項目にアクセスする。

## 画面構成

- List（画面幅の80%）:
  - データセクション: ログイン
  - ヘルプセクション: アプリの使い方、お問い合わせ
  - その他セクション: 利用規約、プライバシーポリシー、アプリバージョン

## 前提条件

- 各タブ画面のメニューボタン（三本線アイコン）タップで表示

## 機能仕様

### メニュー項目

| セクション | 項目 | アイコン | タップ時の動作 |
|-----------|------|---------|--------------|
| データ | ログイン | person.circle | ログイン画面をfullScreenCoverで表示 |
| ヘルプ | アプリの使い方 | questionmark.circle | チュートリアル画面をsheetで表示 |
| ヘルプ | お問い合わせ | envelope | メールアプリが起動 |
| その他 | 利用規約 | doc.text | Safariで利用規約ページを開く |
| その他 | プライバシーポリシー | lock.shield | Safariでプライバシーポリシーページを開く |
| その他 | アプリバージョン | info.circle | バージョン番号を表示（タップ不可） |

### メニュー項目の表示

| 表示要素 | 説明 |
|---------|------|
| アイコン | SF Symbolアイコン |
| タイトル | 項目名 |
| サブタイトル（あれば） | グレーテキストで右寄せ |
| 矢印（サブタイトルなし） | chevron.right（右矢印） |

### お問い合わせメール

メール起動時に自動設定される内容:
| 項目 | 内容 |
|------|------|
| 宛先 | アプリ開発者のメールアドレス |
| 本文テンプレート | デバイス情報、OSバージョン、アプリバージョン |

## 画面遷移

| 遷移元操作 | 遷移先 | 遷移方法 |
|-----------|--------|---------|
| ログイン | ログイン画面 | fullScreenCover |
| アプリの使い方 | チュートリアル画面 | sheet |
| お問い合わせ | メールアプリ | UIApplication open |
| 利用規約 | Safari | UIApplication open |
| プライバシーポリシー | Safari | UIApplication open |

## 補足

- メニューは画面幅の80%で左寄せに表示される
- メールアプリが未設定の場合、エラーメッセージが表示される
